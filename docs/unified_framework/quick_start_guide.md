# Quick Start Guide: Your First 7 Days

**Goal:** Go from idea to validated landing page in 1 week

---

## Day 1: Pre-Validation (2 hours)

### Morning (1 hour)
- [ ] Read Phase 0 checklist
- [ ] Write one-sentence problem statement
- [ ] Rate pain intensity (1-10)

### Afternoon (1 hour)
- [ ] Complete feasibility check (4 dimensions)
- [ ] Check red flags
- [ ] Make GO/NO-GO decision

**Decision Point:** If STOP → Find new problem. If GO → Continue.

---

## Day 2: Market Research (4 hours)

### Morning (2 hours)
- [ ] List 5 direct competitors
- [ ] Find 3 failed competitors
- [ ] Document pricing landscape

### Afternoon (2 hours)
- [ ] Define ICP (Ideal Customer Profile)
- [ ] Create anti-persona
- [ ] Estimate unit economics (CAC, LTV)

**Output:** Competitor matrix + ICP document

---

## Day 3: Smoke Test Prep (4 hours)

### Morning (2 hours)
- [ ] Choose landing page tool (Loveable/Framer/Template)
- [ ] Write headline (10 words max)
- [ ] Design 3-tier CTA strategy

### Afternoon (2 hours)
- [ ] Create Wizard of Oz demo (video/Figma/screenshots)
- [ ] Set up domain ($12)
- [ ] Configure analytics (GA4 + Hotjar)

**Output:** Landing page copy + demo ready

---

## Day 4: Launch Landing Page (4 hours)

### Morning (2 hours)
- [ ] Build landing page
- [ ] Test all CTAs
- [ ] Mobile responsiveness check

### Afternoon (2 hours)
- [ ] Deploy to production
- [ ] Test analytics tracking
- [ ] Share with 5 friends for feedback

**Output:** Live landing page at yoursite.com

---

## Day 5: Initial Traffic (6 hours)

### Morning (2 hours)
- [ ] Email personal network (20-50 people)
- [ ] Post in 3 relevant communities
- [ ] Tweet/LinkedIn post

### Afternoon (4 hours)
- [ ] Send 25 LinkedIn DMs to ICP
- [ ] Find and join 5 more communities
- [ ] Set up Google Ads ($25 test)

**Goal:** 50-100 visitors by end of day

---

## Day 6: User Interviews (6 hours)

### Morning (2 hours)
- [ ] Email everyone who clicked CTA
- [ ] Schedule 5 interviews for next week
- [ ] Prepare interview script

### Afternoon (4 hours)
- [ ] Conduct 2-3 interviews
- [ ] Take detailed notes
- [ ] Document key insights

**Output:** 3 interview transcripts

---

## Day 7: Week 1 Review (2 hours)

### Morning (1 hour)
- [ ] Review metrics dashboard
- [ ] Analyze conversion funnel
- [ ] Read all user feedback

### Afternoon (1 hour)
- [ ] List top 3 insights
- [ ] Plan Week 2 priorities
- [ ] Decide: Continue or pivot?

**Decision Point:** 
- Positive signs (>2% conversion, good interviews) → Continue Week 2
- Weak signals (<1% conversion, "meh" interviews) → Pivot messaging
- No interest (0 conversions, no responses) → New problem

---

## After 7 Days, You Should Have:

- ✅ Validated problem statement
- ✅ Live landing page
- ✅ 100-200 visitors
- ✅ 2-5 CTA conversions
- ✅ 3-5 user interviews
- ✅ Clear next steps

**If YES → Continue to Week 2 (more interviews + traffic)**  
**If NO → Pivot or new idea**
