# User Interview Script (15-20 min)

## Pre-Interview Setup
- [ ] Calendar invite sent with $25 gift card mention
- [ ] Zoom/Google Meet link ready
- [ ] Recording permission requested
- [ ] Notebook/doc ready for notes

## Introduction (2 min)
```
Hi [Name], thanks for taking the time. I'm [Your Name], working on a 
solution for [problem]. I'm not here to pitch you anything - I genuinely 
want to understand your experience with [problem area].

This will take 15-20 minutes. I'll send you a $25 Amazon gift card after 
as a thank you. Mind if I record this for my notes? (wait for yes)

Perfect. Let's start...
```

## Question 1: The Story (5 min)
```
Tell me about the LAST TIME you dealt with [problem].

- When was it?
- What were you trying to accomplish?
- What happened?
- How did it make you feel?
```

**Listen for:**
- Frequency (daily/weekly/monthly)
- Emotional intensity
- Specific details (not hypotheticals)

## Question 2: Current Solutions (3 min)
```
What have you TRIED to solve this problem?

- What tools/services did you use?
- How much did they cost?
- Why did you stop using them (if you did)?
```

**Listen for:**
- Willingness to pay (budget exists)
- Switching costs
- Feature gaps

## Question 3: Quantify the Pain (3 min)
```
What does this problem COST you?

- In time: How many hours per [week/month]?
- In money: Any direct costs?
- In opportunity: What can't you do because of this?
```

**Listen for:**
- Concrete numbers (not "a lot")
- Urgency signals

## Question 4: Workflow Context (4 min)
```
Walk me through your CURRENT WORKFLOW for [task].

- When does this happen in your day?
- What other tools are involved?
- Who else is part of this process?
```

**Listen for:**
- Integration needs
- Existing habits/flows
- Organizational constraints

## Question 5: Switching Conditions (2 min)
```
What would need to be TRUE for you to try a new solution?

- What's your biggest concern?
- What would make you say "take my money"?
- What's a dealbreaker?
```

**Listen for:**
- Price sensitivity
- Technical requirements
- Trust factors

## Closing (1 min)
```
This was super helpful. Last question: would you be open to seeing 
an early version if I build this? (If yes, get commitment)

I'll send that gift card within 24 hours. Thanks again!
```
