# Domain Name Brainstormer

Generates brandable domain names + checks real-time availability.

## When to Use
- Starting new project/company
- Launching product
- Need professional domain quickly
- Rebranding

## Process
1. Understand project (description, audience, keywords)
2. Brainstorm 15-20 creative options
3. Check availability (.com, .io, .dev, .ai, .app)
4. Prioritize best available
5. Provide alternatives

## Naming Strategies
- **Descriptive**: AuctionIntel, PropertyBid
- **Invented**: BidDeed, Propvio
- **Compound**: DealFlow, CaseFinder
- **Modified**: Biddify, Dealr
- **Prefixed**: GetBids, TheBidder
