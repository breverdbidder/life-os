# ADHD Task Management

Tracks tasks and provides interventions for ADHD-optimized productivity.

## Task States
INITIATED → SOLUTION_PROVIDED → IN_PROGRESS → COMPLETED/ABANDONED

## Abandonment Detection
- 0-30 min: Gentle nudge
- 30-60 min: Pattern recognition  
- >60 min: Accountability check

## Interventions
- Micro-commitment: "Just step 1?"
- Body doubling: "I'll wait while you..."
- Chunking: Break into tiny steps

## Usage
Activates automatically on every task request.
