# Claude Skills for Life OS

ADHD-optimized skills for productivity, learning, and family management.

## Installed Skills

1. **frontend-design** - Creates distinctive production-grade UIs
2. **skill-creator** - Builds custom Claude skills  
3. **domain-name-brainstormer** - Generates available domain names
4. **content-research-writer** - Writes content with citations
5. **lead-research-assistant** - Finds qualified leads (great for Michael's D1 recruiting!)
6. **stripe-integration** - Implements Stripe payments

## Usage

Skills auto-activate when relevant:
```
"Use content research writer for Michael's recruiting email"
"Use lead research assistant to find D1 swim programs"
```
