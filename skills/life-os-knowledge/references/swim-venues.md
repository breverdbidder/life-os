# Swim Venues Reference - Michael Shapira

## Florida Competitive Pools

### Ocala Aquatic Center
- Location: Ocala, FL
- Pool Speed: FAST ⚡
- Pool Type: 50m outdoor (can be configured 25y)
- Preferred Hotel: Staybridge Suites Ocala
- Drive from Satellite Beach: ~2.5 hours (142 miles)
- Parking: Ample, free
- Notes: Known for fast times, good competition venue

### Satellite High School Pool
- Location: Satellite Beach, FL
- Pool Speed: Standard
- Pool Type: 25y
- Notes: Home pool, practice facility

---

## Pool Speed Guide
- **FAST** ⚡ - Deep water, good gutters, minimal turbulence
- **STANDARD** - Normal competition pool
- **SLOW** - Shallow, older design, more turbulence

## Adding New Venues

2. Pool configuration (25y, 25m, 50m)
