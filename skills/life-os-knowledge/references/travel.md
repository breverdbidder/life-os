# Travel Routes Reference - Shapira Family

## Common Swim Meet Routes

### Satellite Beach → Ocala
- Distance: 142 miles
- Duration: ~2.5 hours
- Route: I-95 N → SR-40 W → Ocala
- Stops: None needed, straight shot
- Notes: Leave by 4:15 PM for 7 PM arrival

### Satellite Beach → Gainesville
- Distance: ~160 miles
- Duration: ~2.5-3 hours
- Route: I-95 N → SR-40 W → I-75 N
- Notes: Similar to Ocala route

### Satellite Beach → Jacksonville
- Distance: ~175 miles
- Duration: ~3 hours
- Route: I-95 N direct
- Notes: Can be traffic near Jacksonville
